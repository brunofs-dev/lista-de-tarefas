# Aplicativo Lista de Tarefas

Este é um aplicativo simples de Lista de Tarefas construído com HTML, CSS e JavaScript.

## Funcionalidades

- Adicionar novas tarefas
- Marcar tarefas como concluídas
- Excluir tarefas

## Instalação

Para executar este projeto localmente, basta clonar o repositório e abrir `index.html` no seu navegador da web.

```bash
git clone <url-do-repositorio>
cd lista-de-tarefas
# Em seguida, abra index.html no seu navegador
```

## Uso

1. Abra `index.html` no seu navegador da web.
2. Digite uma tarefa no campo de entrada.
3. Clique no botão "Adicionar Tarefa" ou pressione Enter para adicionar a tarefa.
4. Clique em uma tarefa para marcá-la como concluída/incompleta.
5. Clique no botão "X" ao lado de uma tarefa para excluí-la.

## Estrutura do Projeto

- `index.html`: O arquivo HTML principal para a estrutura do aplicativo.
- `style.css`: Contém a estilização para o aplicativo.
- `index.js`: Contém a lógica JavaScript para a funcionalidade da Lista de Tarefas.
- `README.md`: Este arquivo.
